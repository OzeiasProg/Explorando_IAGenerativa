#Introdução

Este relatório apresenta o processo de implementação de um código em python de um exercício de ETL ( Extract, Transform e Load) com interação com uma ferramenta de IA para gerar mensagens de marketing para os clientes de uma loja de calçados, cujo objetivo e trazer ele de volta para loja para comprar mais calçados.

#Metodologia do Trabalho

Foi criado uma lista de cliente salvos no mesmo diretório do código no formato csv, chamado “clientes.csv”, no caso específico com 03 clientes devido as limitações de interação ferramenta de IA na opção de conta gratuita.
A ferramenta escolhida foi do Gemini IA  da Google, mas o código pode ser adaptado para outras opções de ferramentas de IA inclusive com planos pagos que não terão as limitações de interações.
O código em python foi realizado no Vscode mas pode ser executado em outros softwares python disponíveis no mercado.

#Instruções para Reprodução

Criação de uma lista simplificada de clientes com “id” e “nome”; 

Cadastro de uma conta de uma ferramenta de IA; 

Instalação de todas as dependências ( extensões de python necessárias) em função da ferramenta de IA escolhida; 

Fornecimento da chave de acesso “api_key” da ferramenta de IA;

Rodar o programa e checar o resultado que retorna uma frase de marketing para cada cliente do arquivo csv.
